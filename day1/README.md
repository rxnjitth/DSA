# Day 1 - Number Programs

This folder contains four Python programs that work with numbers and their properties.

## Programs Overview

### 1. Automorphic Number (`automorphic.py`)
**What it does:** Checks if a number is automorphic.

**What is an Automorphic Number?**
A number whose square ends with the number itself.

**Examples:**
- 5 is automorphic because 5² = 25 (ends with 5)
- 76 is automorphic because 76² = 5776 (ends with 76)
- 10 is NOT automorphic because 10² = 100 (doesn't end with 10)

### 3. Harshad Number (`harshad_number.py`)
**What it does:** Checks if a number is a Harshad number.

**What is a Harshad Number?**
A number that is divisible by the sum of its digits.

**Examples:**
- 18 is a Harshad number because 18 ÷ (1+8) = 18 ÷ 9 = 2 ✓
- 12 is a Harshad number because 12 ÷ (1+2) = 12 ÷ 3 = 4 ✓
- 19 is NOT a Harshad number because 19 ÷ (1+9) = 19 ÷ 10 = 1.9 ✗

### 4. Strong Number (`strong_number.py`)
**What it does:** Checks if a number is a strong number.

**What is a Strong Number?**
A number that equals the sum of the factorials of its digits.

**Examples:**
- 145 is a strong number because 1! + 4! + 5! = 1 + 24 + 120 = 145 ✓
- 1 is a strong number because 1! = 1 ✓
- 2 is a strong number because 2! = 2 ✓
- 10 is NOT a strong number because 1! + 0! = 1 + 1 = 2 ✗
