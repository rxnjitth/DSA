# Day 3 - Number and String Checks

This folder contains two Python programs for basic number and string checks.

## Programs Overview

### 1. First Positive Number (`first_positive_num.py`)
**What it does:** Finds the first positive number from a list.

**Example:**
- Input: `[-3, -2, 0, 5, 8]`
- Output: `5`

---

### 2. Palindrome (`palindrome.py`)
**What it does:** Checks whether a word or number reads the same forward and backward.

**Examples:**
- `madam` -> Palindrome
- `racecar` -> Palindrome
- `hello` -> Not a palindrome
