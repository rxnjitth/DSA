# Day 2 - Array Programs

This folder contains two Python programs focused on common array problems.

## Programs Overview

### 1. 3Sum (`3sum.py`)
**What it does:** Finds triplets in an array whose sum is zero.

**Example:**
- Input: `[-1, 0, 1, 2, -1, -4]`
- Output: `[[-1, -1, 2], [-1, 0, 1]]`

---

### 2. Move Zeros (`move_zeros.py`)
**What it does:** Moves all zeros in an array to the end while keeping the order of non-zero elements.

**Example:**
- Input: `[0, 1, 0, 3, 12]`
- Output: `[1, 3, 12, 0, 0]`
